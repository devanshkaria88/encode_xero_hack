# Live-run transcripts

Two interchangeable sets, one folder each. Every set contains:

- `civicpulse-advisory-call.txt` — paste into the **CivicPulse advisory call** meeting (existing client, autonomy ON). Each version contains exactly one billable extra: about 15 minutes of review work done after the previous call and explicitly never invoiced. Robyn prices it as one 30-minute minimum block at £50/h (£25, Clause 3) on top of the meeting time, then sends the invoice through Xero with zero clicks.
- `northbeam-intro-call.txt` — paste into the **Northbeam intro call** meeting (unknown attendee). A discovery call with nothing billable; Robyn queues devansh8801@gmail.com as a potential client and starts watching that sender for the signed agreement email.

`set-1` is the pair from the runbook (rehearsed baseline). `set-2` is a fresh variant with the same beats, so a rehearsal and the live demo never reuse the same words. The full order of operations is in `../RUNBOOK-live-litmus.md`.
